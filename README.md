## Project Summary
This repository organizes materials for the DSA 5900 practicum project studying whether the Joe's Crab Shack no-tipping policy intervention was associated with measurable changes in customer review language, ratings, attribute-level sentiment, or comparative outcomes.

## Phase Structure
1. Phase 1: Comparison Frame
2. Phase 2: Data Assembly
3. Phase 3: Attribute Framework
4. Phase 4: Attribute Scoring
5. Phase 5: Comparative Analysis

## Core Project Materials
The repository is intended to organize the main project materials used for the final report, slide deck, and oral presentation, including:

- Phase completion reports
- Mermaid diagrams for simplified and detailed project workflows
- Comparison-frame and data-assembly documentation
- Attribute-framework and attribute-scoring documentation
- Comparative-analysis materials
- Final report and presentation materials

## Repository Map
- `00_project_foundation/`: minimal holding area for high-level project context that does not belong to a specific phase.
- `01_phase_1_comparison_frame/`: treatment/control/competitor design and manifest logic.
- `02_phase_2_data_assembly/`: Yelp extraction, cleaning, and master dataset construction.
- `03_phase_3_attribute_framework/`: human benchmark, 97-attribute framework, and canonical 40-attribute materials.
- `04_phase_4_attribute_scoring/`: multi-model attribute identification and scoring pipeline.
- `05_phase_5_comparative_analysis/`: statistical, machine-learning, and comparative analysis.
- `06_figures_and_diagrams/`: Mermaid source files and visual outputs.
- `07_reports_and_presentations/`: proposal, progress report, final report, slide deck, and oral presentation materials.
- `08_project_admin/`: syllabus, submission records, AI-use notes, cost records, and coordination files.
- `archive/`: exploratory, superseded, or deprecated materials.
