## Project Summary
This repository documents the DSA 5900 practicum project studying whether the Joe's Crab Shack no-tipping policy intervention was associated with measurable changes in customer review language, ratings, attribute-level sentiment, or comparative outcomes.

## Phase Structure
1. Phase 1: Comparison Frame
2. Phase 2: Data Assembly
3. Phase 3: Attribute Framework
4. Phase 4: Attribute Scoring
5. Phase 5: Comparative Analysis

## Current Core Artifacts
- `location_manifest_final.csv`: locked 61-location operational manifest.
- `consolidated_cleaned_reviews.csv`: final Phase 2 review-level dataset with 16,997 rows × 42 columns.
- Final 40-attribute canonical framework.
- Mermaid diagrams for simplified and detailed project workflows.
- Completion reports for Phases 1–5.

## Repository Map
- `00_project_foundation/`: project summary, sponsor context, proposal context, literature notes.
- `01_phase_1_comparison_frame/`: treatment/control/competitor design and manifest logic.
- `02_phase_2_data_assembly/`: Yelp extraction, cleaning, and master dataset construction.
- `03_phase_3_attribute_framework/`: human benchmark, 97-attribute framework, canonical 40 attributes.
- `04_phase_4_attribute_scoring/`: multi-model attribute identification and scoring pipeline.
- `05_phase_5_comparative_analysis/`: statistical, machine-learning, and comparative analysis.
- `06_figures_and_diagrams/`: Mermaid source files and visual outputs.
- `07_reports_and_presentations/`: proposal, progress report, final report, slide deck, and oral presentation materials.
- `08_project_admin/`: syllabus, submission records, AI-use notes, cost records, coordination files.
- `archive/`: exploratory, superseded, or deprecated materials.
